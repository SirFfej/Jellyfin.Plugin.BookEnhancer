BookEnhancers Plugin Configuration

Metadata Sources

Enrich your book library with metadata from online sources. The plugin extracts ISBN from book files and looks up full metadata from the sources below, cascading from highest quality to fallback.

Hardcover

Primary source — best quality metadata including authors, narrators, translators, illustrators, series, genre tags, ratings, ASIN, and more. Requires a free API key.

Paste your Hardcover API key here.
How to get a Hardcover API Key
  1. Go to https://hardcover.app/account/api
  2. Sign in to your Hardcover account (or create one — it's free)
  3. Click Generate API Key
  4. Copy the generated key and paste it in the field above
  5. Rate limit: 60 requests per minute, free
  6. Note: API tokens expire annually on January 1st

Google Books

Fallback source — good metadata coverage. Optional API key increases rate limit to 1,000 requests/day.

Leave blank to use the unauthenticated endpoint (IP-based rate limiting). Get a key at Google Cloud Console (enable Books API).

OpenLibrary

Final fallback — free, no API key required. Covers many public domain and older titles not found in other sources.

Format Grouping

Group multiple formats (EPUB, PDF, audiobook, comic) of the same book under a single library entry. Library counts reflect unique titles, not files. Books are matched by ISBN when available.

How to determine if two files represent the same book.
Merge existing duplicate entries into grouped entries. Run this after a library scan.

Managed Directories

Configure external folders to scan for book files. Files are moved into your Jellyfin library organized as Author / Book Title / file. Each source folder can target a different library folder (e.g., separate folders for print books and audiobooks).

Immediately scan all enabled source directories and move files into the library.
How often to automatically scan source directories. Set to 0 to disable.

General Settings

Minimum confidence score (0.0–1.0) for fuzzy title matching. Higher values require closer matches. Default: 0.85.
