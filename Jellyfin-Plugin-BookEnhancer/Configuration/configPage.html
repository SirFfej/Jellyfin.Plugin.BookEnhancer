Metadata Sources

Enrich your book library with metadata from online sources. The plugin extracts ISBN from book files and looks up full metadata from the sources below, cascading from highest quality to fallback.

Hardcover

Primary source — best quality metadata including authors, narrators, translators, illustrators, series, genre tags, ratings, ASIN, and more. Requires a free API key.

Paste your Hardcover API key here.
How to get a Hardcover API Key
  1. Go to hardcover.app/account/api
  2. Sign in to your Hardcover account (or create one — it's free, no subscription required)
  3. Click Generate API Key
  4. Copy the generated key and paste it in the field above
  5. Rate limit: 60 requests per minute, free tier
  6. Note: API tokens expire annually on January 1st — generate a new one each year
A Hardcover API key is strongly recommended — it provides the best quality metadata (authors, narrators, series, genres). The plugin works without it by falling back to Google Books and OpenLibrary.

Google Books

Fallback source — good metadata coverage.

You do NOT need an API key — the plugin works without one (IP-based rate limiting applies). If you have a key: Get one at Google Cloud Console , enable the Books API, then paste it here for a higher rate limit (1,000 requests/day).

OpenLibrary

Final fallback — free, no API key required. Covers many public domain and older titles not found in other sources.

Comic Vine

Comic-specific metadata source — ideal for single-issue comics. Provides creative team (penciller, inker, colorist, etc.), characters, story arcs, cover art, and publisher info. Free API key required.
